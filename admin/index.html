Site Admin

Sign in with your GitHub personal access token to manage content.

How to get a GitHub token
  1. Go to github.com → Settings → Developer Settings
  2. Click Personal access tokens → Fine-grained tokens
  3. Click Generate new token
  4. Under Repository access choose livewiremarkets1/lwl-2026-prelaunch
  5. Under Permissions → Repository permissions → Contents choose Read and write
  6. Generate and paste the token above
